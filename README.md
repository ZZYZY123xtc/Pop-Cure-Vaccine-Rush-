# 🦠 萌菌大作战 - Pop Cure Vaccine Rush

> "Don't fight them. Cure them." 一款基于 HTML5 Canvas 的 Roguelite 休闲策略网页游戏。

## 游戏简介

萌菌大作战是一款对抗"指数级增长"的点击策略游戏。玩家扮演免疫指挥官，通过消除变异病毒收集 DNA，并在病毒吞噬世界前研发出疫苗。

不同于传统的无尽点击游戏，本项目引入了：
- 🎲 **Roguelike 技能成长系统** - 每局游戏随机获得不同的技能升级
- 🗺️ **关卡地图推进机制** - 逐步解锁更高难度的关卡
- 💉 **疫苗清屏机制** - 策略性使用疫苗清除所有病毒

在轻度休闲的外壳下，保留了紧张刺激的策略博弈体验。

## 游戏特性

### 核心玩法
- **点击消灭病毒**：点击屏幕上的病毒来攻击它们
- **收集 DNA**：消灭病毒后获得 DNA 资源
- **指数级增长**：病毒会随时间自动增殖变大
- **疫苗研发**：积累足够 DNA 可使用疫苗清屏

### Roguelike 技能升级
- 💪 **点击威力**：提升每次点击的伤害值
- 🧬 **DNA加成**：增加消灭病毒获得的 DNA
- ⏰ **减缓增殖**：降低病毒的增长速度
- ⚡ **自动伤害**：持续对所有病毒造成伤害

### 关卡系统
- 每关需要消灭一定数量的病毒才能晋级
- 关卡越高，病毒越强大，生成速度越快
- 病毒数量达到上限则游戏失败

## 如何开始

1. 克隆或下载此仓库
2. 用浏览器打开 `index.html` 文件
3. 点击"开始游戏"按钮
4. 点击病毒来消灭它们！

## 游戏策略提示

1. **优先升级点击威力**：在游戏初期快速消灭病毒
2. **合理使用疫苗**：在病毒过载前使用疫苗清屏
3. **平衡资源分配**：不要把所有 DNA 都花在一个升级上
4. **注意病毒增长**：病毒会指数级增长，要及时消灭

## 开发说明

本项目采用纯前端技术开发，无需后端服务器即可运行。所有游戏逻辑都在客户端完成。
